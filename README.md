## 🧾 서랍
<h3>새로운 서비스 / 신뢰 / 고급화 / 윤리적인 소비 / 환경보호</h3>
나의 기억이 당신의 기억으로, 그리고 우리의 기억으로 바뀔 때

**가성비(價性比)** 보다 **가심비(價心比)** <br/>
다시 돌아온 아나바다 운동, <br/>
필환경 시대에 동참하는 제로 웨이스트 <br/>
착한소비, 윤리적인 소비를 지향하는 소비 트랜드 <br/>
'중고거래'를 하지 않을 이유는 없습니다.

구매자는 거래 준비 과정, 판매자는 신뢰성을 <br/>
중고거래 플랫폼 사용자들의 불편함을 대신 할 <br/>
**"기억명세서"** <br/>
사용하지 않을 이유가 있을까요?
